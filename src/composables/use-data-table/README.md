# 表格

## 列头格式

```json
[
    {
        "key":"string : 字段名",
        "title":"string : 列头标题",
        "sortName":"string : 排序名称",
        "align":"[left | right | center] : 对齐方式，默认是 left",
        "isSort":"boolean : 是否使用排序",
        "defaultSort":"[descend | ascend] : 默认排序方式",
        "fixed":"[left | right] : 列头定位位置"
        "width":"number : 列宽"
    }
]
```
